## Descrição
O projeto **De Novo Não!** é um sistema desenvolvido para otimizar o atendimento e a gestão de pacientes da ONG Turma do Bem (TDB).  
O foco desta Sprint 02 foi implementar **responsividade**, garantindo uma experiência de usuário consistente em diferentes dispositivos, além de incluir **interatividade em JavaScript** e validação de formulários.

---

## Tecnologias Utilizadas
- **HTML** — Estrutura semântica do conteúdo e navegação.
- **CSS** — Estilização completa e responsividade usando media queries.
- **JavaScript** — Interatividade (menu hambúrguer, validação de formulários, localStorage e sessionStorage).
