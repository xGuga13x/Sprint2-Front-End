Perguntas Frequentes

Confira as dúvidas mais comuns sobre o projeto De Novo Não! e entenda como ele transforma o atendimento odontológico social.

O que é o projeto "De Novo Não!"?

É um sistema criado para otimizar o registro e o armazenamento das informações dos pacientes atendidos pela Turma do Bem (TDB). Ele combate a perda de dados e o retrabalho, promovendo um atendimento mais eficiente, seguro e humanizado.

Qual problema o sistema resolve?

O sistema soluciona a desorganização e perda de informações em atendimentos odontológicos, garantindo um histórico contínuo e confiável dos pacientes.

Quem pode utilizar o sistema?

O "De Novo Não!" foi desenvolvido para dentistas e voluntários da ONG Turma do Bem, mas também pode ser adaptado para outras instituições sociais.

Quais tecnologias foram utilizadas?

O projeto utiliza Python para a lógica e chatbot, Java para modelagem UML, banco de dados relacional para o armazenamento das informações e HTML, CSS e JavaScript para o front-end.

O que faz o Chatbot Assistente Odontológico?

O chatbot auxilia no preenchimento correto dos dados, lembra campos obrigatórios e responde dúvidas, agilizando o atendimento odontológico.

Como o sistema garante a segurança e confiabilidade dos dados?

Cada paciente possui um identificador único, com validações e um banco de dados estruturado para prevenir duplicações e inconsistências.

O projeto "De Novo Não!" é fruto da parceria entre a FIAP e a Turma do Bem, unindo tecnologia e impacto social.
